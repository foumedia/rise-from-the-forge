How to Use This Site

This site is designed for quick reference during play. Use the navigation buttons above to jump directly to specific rule sets. Each page is mobile-friendly and includes links back to this home page.

New players should start with the Core Rules. If you're in combat, jump straight to the Combat page. Game masters can check Factions and Quests for narrative hooks and updates.

World of the Fractured Realms

The cataclysm tore the continents asunder, but in the voids between, new powers rose. Skyfaring cities float above scorched valleys, while ancient machines awaken below. Whether you’re a relic hunter, an arcane renegade, or a soldier of the shifting alliances, the world is yours to explore — or burn.

Take a moment to read the lore, explore the factions, and immerse yourself in the living history of the campaign. The world is vast, and not even the stars can show you all its secrets.
