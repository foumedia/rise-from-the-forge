House Rules

Officially posted house rules

Rule
Description
Rule 1
When playing a game, if it is either Player's First Game in the Campaign, instead of rolling on the Scenario Table, play in the Wasteland. A player starting a new Crew does not get this benefit.
Rule 2
After playing a Game, if it is either Player's First Game in the Campaign, do not roll for Consequences or Injuries. A player starting a new Crew does not get this benefit.
Rule 3
Unofficial Faction lists are permitted with Overseer Approval.
