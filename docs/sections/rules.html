House Rules

Officially posted house rules

Rule
Description
Rule 1
When playing a game, if it is either Player's First Game in the Campaign, instead of rolling on the Scenario Location Table, play in the Wasteland. New Crews for an existing Player, do not trigger this.
Rule 2
After playing a Game, if it is either Player's First Game in the Campaign, both players do not roll on the Aftermath Table for Consequences or Injuries. New Crews for an existing Player, do not trigger this.
Rule 3
Unofficial Faction lists are permitted with Overseer Approval.
Rule 3
No Wild Wasteland rules are currently in play
Rule 4
Annihilation Scenarios will not currently trigger. This rule will change when we have a satisfactory solution.
Rule 5
If you already have a Unit with 'Four Leaf Clover' Perk in your Crew, you may not choose it as a Perk during Crew Training.
